# agents

A repository for GitHub Copilot agents — configurations, instructions, and resources for working with [GitHub Copilot coding agent](https://docs.github.com/en/copilot/using-github-copilot/using-copilot-coding-agent-to-work-on-tasks).

## Overview

This repository is set up to work seamlessly with GitHub Copilot's coding agent. The Copilot coding agent can autonomously work on issues, implement features, fix bugs, and open pull requests.

## Getting Started

Assign any open issue to **Copilot** to have the coding agent work on it automatically.

## Repository Configuration

- **`.github/copilot-instructions.md`** — Custom instructions that guide the Copilot coding agent when working in this repository.
